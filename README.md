# The-Ultimate-Webshop
# SuperCar Vault

SuperCar Vault is een website waar je exclusieve supercars en hypercars kan bekijken.  
Op de homepage zie je een overzicht van verschillende wagens met een afbeelding, prijs en label (Nieuw, Sale of Limited).  
Als je op “Bekijk details” klikt, kom je op een detailpagina met extra informatie over de auto.

De website is volledig responsive, dus ze werkt goed op zowel computer als smartphone.

Er is ook een eenvoudige admin-omgeving.  
Daar kan je inloggen op een demo dashboard en zie je een lijst van producten.  
Je kan er ook een formulier openen om zogezegd een nieuw product toe te voegen.  
Dit is enkel front-end en werkt zonder echte database.

Plan van aanpak en timing

Ik ben begonnen met een eerste idee voor mijn website.
Eerst maakte ik een webshop rond parfums, maar ik merkte dat ik de productfoto’s van een externe website moest halen en dat deze niet bruikbaar waren.
Daarom heb ik besloten om van thema te veranderen.

Ik koos een nieuw thema: een webshop rond supercars.
Ik maakte de eerste versie van de index.html en begon al met de HTML van de 6 productpagina’s.
Ook zocht ik geschikte afbeeldingen en plaatste deze in mijn projectmap.

Ik werkte de homepage verder uit en voltooide alle productdetailpagina’s.
Daarna begon ik aan de frontend CSS: layout, kleuren, fonts en responsive design.
Ook maakte ik de eerste adminpagina’s (login, dashboard, product-form).

Ik ontwierp een apart CSS-thema voor de adminomgeving.
Daarna werkte ik het dashboard en formulier verder af.
Tot slot controleerde ik de responsiviteit en verbeterde ik kleine foutjes.
