# SuperCar Vault

SuperCar Vault is een website waar je exclusieve supercars en hypercars kan bekijken.  
Op de homepage zie je een overzicht van verschillende wagens met een afbeelding, prijs en label (Nieuw, Sale of Limited).  
Als je op “Bekijk details” klikt, kom je op een detailpagina met extra informatie over de auto.

De website is volledig responsive, dus ze werkt goed op zowel computer als smartphone.

Er is ook een eenvoudige admin-omgeving.  
Daar kan je inloggen op een demo dashboard en zie je een lijst van producten.  
Je kan er ook een formulier openen om zogezegd een nieuw product toe te voegen.  
Dit is enkel front-end en werkt zonder echte database.
